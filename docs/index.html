About the Project

PowerLauncher is a free, open-source Minecraft launcher for Windows. It provides a clean dark interface, built-in mod marketplace powered by Modrinth, automatic Java runtime management, and full game lifecycle control including crash detection.

The launcher stores all files in its own directory and does not interfere with the official Minecraft launcher or .minecraft folder.
