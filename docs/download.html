The installer includes all dependencies. No additional .NET or Java installation required (Java is installed automatically on first launch).

System Requirements

OSWindows 10 or Windows 11
Architecturex64 (64-bit)
Disk Space~200 MB (launcher + Java runtime)
RAM4 GB minimum (8 GB recommended for modded)
InternetRequired for downloading game files and mods

Build from Source

git clone https://github.com/nqox3/PowerLauncher.git
cd PowerLauncher
dotnet restore
dotnet build
dotnet run --project MinecraftLauncher

Requires .NET 8 SDK installed on your machine.
