Installation

  1. Download the installer PowerLauncher-Setup-1.0.0.exe from the Download page
  2. Run the installer and follow the on-screen instructions
  3. On first launch, the launcher will create directories and check for Java
  4. If Java is not found, it will be downloaded automatically (Java 21, Adoptium)

First Launch

  1. Enter a player name in the Username field and click the + button to create an account
  2. Click "Refresh Versions" to load the version list from Mojang servers
  3. Select a version from the dropdown (only releases shown by default)
  4. Click LAUNCH MINECRAFT
  5. The launcher will download the required files and start the game
  6. The launcher window hides while the game is running
  7. When the game closes, the launcher reappears automatically

Accounts

PowerLauncher supports offline (cracked) accounts. Each account has a username and a generated UUID.

  • Click + to add a new account with the entered username
  • Click - to remove the currently selected account
  • Select an account from the dropdown to switch between them
  • Accounts are saved in %AppData%/PowerLauncher/accounts.json

Mod Market

The Mod Market tab lets you search and install mods from Modrinth directly.

  1. Navigate to the "Mod Market" tab
  2. Enter a mod name in the search box and press Enter or click Search
  3. Use the loader filter to show only Fabric, Forge, Quilt, or NeoForge mods
  4. Sort results by relevance, downloads, updated, or newest
  5. Click "Install" next to any mod to download it
  6. The mod jar file is saved to the game's mods/ folder

Note: You need a mod loader (Fabric, Forge, etc.) installed as a game version to use mods. Vanilla Minecraft does not load mods.

Java Runtime

PowerLauncher manages Java automatically. On first launch or when Java is missing, it downloads Java 21 from Adoptium (Eclipse Temurin).

  • Java is installed to %AppData%/PowerLauncher/runtime/java-21/
  • The Java tab shows current status, version, and path
  • Click "Install/Reinstall Java 21" to force a fresh download
  • You can also set a custom Java path in Settings

File Structure

%AppData%/PowerLauncher/
+-- settings.json          - launcher configuration
+-- accounts.json          - saved player accounts
+-- game/                  - Minecraft game files
|   +-- versions/          - installed game versions
|   +-- assets/            - game resources
|   +-- libraries/         - Java libraries
|   +-- mods/              - installed mods (.jar files)
|   +-- crash-reports/     - game crash logs
+-- runtime/
    +-- java-21/           - auto-installed Java runtime

All data is stored separately from the official .minecraft folder. Uninstalling PowerLauncher removes this directory.

FAQ

The launcher does not start

Make sure you are running Windows 10/11 x64. Try running as administrator. Check if antivirus is blocking the executable.

The game crashes on launch

Check the Console tab for error details. Try increasing Maximum RAM in settings. Disable conflicting mods one by one.

Mods are not working

Vanilla Minecraft does not load mods. You need to install a Fabric or Forge version first, then place mods in the mods folder.

How to update Java?

Go to the Java tab and click "Install/Reinstall Java 21". This will download the latest build.

Where are my saves?

Game saves are in %AppData%/PowerLauncher/game/saves/

Can I use this with the official launcher?

Yes. PowerLauncher uses its own directory and does not touch .minecraft at all.
