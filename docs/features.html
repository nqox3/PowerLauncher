Features

Everything you need to play Minecraft your way

M
Mod Market Search and install mods from Modrinth. Filter by Fabric, Forge, Quilt, or NeoForge. Sort by relevance, downloads, or date.
J
Auto Java Install No Java? The launcher automatically downloads and installs Java 21 (Adoptium). Manage Java from the dedicated tab.
V
All Versions Launch any Minecraft release version. Assets, libraries, and game files are downloaded automatically.
+
Mod Manager Enable, disable, or delete mods with one click. Add mods from local files or the built-in market.
!
Crash Detection If Minecraft crashes, the launcher reopens and shows you what went wrong with the crash report.
U
Offline Accounts Play offline with any username. Add and remove multiple accounts. Player head loaded from CDN.
L
Multi-Language Automatically syncs with your system language. English and Russian supported out of the box.
D
Dark Modern UI Built with WPF UI (Fluent Design). Dark theme, smooth navigation, responsive layout.
S
Skins Preview View player avatars from mc-heads.net. Enter any username to see their skin.
C
Game Console View real-time game output and logs. Copy or clear the console at any time.
